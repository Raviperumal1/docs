AI Video Generation allows users to create short videos from text prompts, images, files, and structured scenes using generative AI models.
This feature is designed for users who want to transform ideas, visuals, or scripts into animated video content with minimal effort.

This document explains **every step of the process**, from uploading inputs to selecting models and generating final videos.

---

## What You Can Do with AI Video Generation

Using AI Video Generation, you can:

- Generate videos from text prompts
- Convert images into animated video clips
- Create scene-based storytelling videos
- Choose different AI models for visual styles
- Iterate and regenerate videos
- Download and reuse generated videos

## Supported Input Types

You can generate videos using the following inputs:

- Text prompts or scripts
- Uploaded images (JPG, PNG, GIF)
- Uploaded documents (PDF, DOCX, TXT) as context
- Screenshots of visual references
- Extracted content from webpages

Inputs can be used individually or combined.

## Step-by-Step: How to Generate a Video


## Step 1: Choose an AI Model

Before generating a video, select an AI model.

### How to Select a Model

1. Click the **Model Selector** at the top
2. Choose an available AI model
3. Confirm your selection

### Model Selection Guidelines

Different models may specialize in:

- Cinematic motion
- Stylized or animated visuals
- Concept art videos
- Realistic environments

Choose a model based on:
- Desired visual quality
- Style preference
- Generation speed

If unsure, start with the default recommended model.

## Preset Prompts for Video Generation

Preset Prompts for Video Generation allow users to create videos instantly by clicking predefined prompt cards.
These presets remove the need for manual typing and help users generate high-quality videos with a single click.

Preset prompts are optimized for common video styles such as cinematic scenes, animations, product shots, and social media content.

---

### How Preset Video Prompts Work

1. User opens the **Video Generation** screen
2. Preset prompt cards are displayed
3. User clicks a preset card
4. The prompt is automatically applied
5. Video generation starts (or is ready to generate)

### How to Download a Generated Video

2. Wait for the generation process to complete
3. Preview the generated video
