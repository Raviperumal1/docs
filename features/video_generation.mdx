# Preset Prompts for Video Generation

Preset Prompts for Video Generation allow users to create videos instantly by clicking predefined prompt cards.
These presets remove the need for manual typing and help users generate high-quality videos with a single click.

Preset prompts are optimized for common video styles such as cinematic scenes, animations, product shots, and social media content.

---

## How Preset Video Prompts Work

1. User opens the **Video Generation** screen
2. Preset prompt cards are displayed
3. User clicks a preset card
4. The prompt is automatically applied
5. Video generation starts (or is ready to generate)

## What Happens When You Click a Preset

When a preset prompt card is clicked:

- The video generation starts immediately
- Default settings are applied:
  - Selected AI model
  - Optimized duration
  - Matching aspect ratio

## How to Download a Generated Video

1. Generate a video using text, image, scene, or preset prompts
2. Wait for the generation process to complete
3. Preview the generated video
4. Click the **‚¨á Download Video** button
5. The video file is saved to your device

## Available Download Options

Depending on the selected model and settings, the following options may be available:

- **Standard Quality Download**
- **High Quality Download**
- **Original Aspect Ratio**
- **Platform-Optimized Format (if enabled)**

---

## Supported Video Formats

Generated videos can be downloaded in the following formats:

- MP4 (default)
- WebM *(if supported by the model)*
