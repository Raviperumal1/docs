## Chat Interface Features

![dashboard](/images/chat_dashboard.png)

### **Copy Response**
Click the **Copy** button below any AI response to copy it to your clipboard.

**Use Cases:**
- Save useful information
- Paste into documents
- Share with others

### **Edit Response**

Click the **Edit** button below any AI response to modify the original prompt and regenerate the response.

When editing is enabled, an input area opens with **Cancel** and **Send** actions.

---

### How Edit Response Works

1. Click the **Edit** button under an AI response
2. The original prompt appears in an editable input field
3. Update or refine the text as needed
4. Choose an action:
   - **Cancel** ‚Äì Discards changes and closes the editor
   - **Send** ‚Äì Submits the edited text and regenerates a new response

---

### What Happens When You Click Send

- The edited text is sent as a new prompt
- The AI generates a fresh response
- The previous response will be clear
- The new response appears below as an updated result

---

### Use Cases

- Refine or clarify a previous request
- Correct mistakes in the original prompt
- Add more details for better results
- Regenerate responses without starting a new chat
- Iterate quickly on the same idea

---

### Tips

- Be specific when editing for improved accuracy
- Add context instead of replacing the entire prompt
- Use short edits for faster iterations
- Cancel if you want to keep the original response unchanged


### **Quote Response**

**How It Works:**
1. Click **Quote** on any response

## Frequently Asked Questions

### How do I change AI models?
Click the model name at the top of the chat and select a different one from the dropdown.

### Can I upload multiple files at once?
Yes, you can upload multiple files in a single conversation. Each file can be referenced separately.

### How long are chats saved?
All your chats are saved indefinitely until you delete them manually.

### Can I edit my messages after sending?
Yes, click the edit icon next to your message to modify it.

### What's the difference between models?
Different models have different strengths. Try asking the same question to multiple models to see which works best for your needs.

### Is there a message limit?
Free plan has daily limits. Pro plans offers unlimited messages. Check your plan details for specifics.

### Can I export my conversations?
Yes, use the export feature in chat history to save conversations as text or PDF.

### How do I delete a chat?
Open the chat history, find the conversation, and click the delete icon.
