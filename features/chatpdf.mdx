# Chat with PDF (ChatPDF)

The **ChatPDF** feature allows you to interact with your entire knowledge-base—documents, webpages, audio or video—as though you’re chatting with them. Instantly analyse, summarise, deep-research, or translate your files. :contentReference[oaicite:1]{index=1}

<Info>
Use this guide to get started with uploading files, managing them, engaging in deep chat sessions, summarising and translating your content.
</Info>

---

## 1. Uploading Files and Starting Chats

### 1.1 Uploading Files
You can import content via:
- Local upload: select one or more files from your computer. :contentReference[oaicite:2]{index=2}
- URL link: paste a web link to analyze and import web content. :contentReference[oaicite:3]{index=3}
- Record note: create an audio or screen-recorded note and add it to your knowledge base. :contentReference[oaicite:4]{index=4}

#### Supported file formats
Documents: PDF, Word, TXT, PPT.
Other types: Webpages, Images, Audio, Video. :contentReference[oaicite:5]{index=5}

#### Free vs Paid plan limits
Be aware that different plan tiers have restrictions on upload size, file count and usage. :contentReference[oaicite:6]{index=6}

### 1.2 Chatting with Multiple Files
- Choose an AI model and select target files to start chatting. :contentReference[oaicite:7]{index=7}
- Mix sources: combine files, web sources and notes into one chat session. :contentReference[oaicite:8]{index=8}
- Deep-Research mode: perform deep analysis across all your selected files. :contentReference[oaicite:9]{index=9}

---

## 2. File Translation
- Open a file in the platform, select your AI model, choose target language (50+ supported). :contentReference[oaicite:10]{index=10}
- The layout, structure and formatting aim to be preserved where possible. :contentReference[oaicite:11]{index=11}

## 3. File Summary
Hover over a file in the file list panel to instantly preview a summary of its content. :contentReference[oaicite:12]{index=12}

## 4. File Management
### 4.1 Search
Use keywords to locate files quickly in your workspace. :contentReference[oaicite:13]{index=13}
### 4.2 Filter
Filter files by type (documents, images, audio, etc.). :contentReference[oaicite:14]{index=14}
### 4.3 Bulk Actions
Move, download or delete multiple files at once. Group related files (e.g., by topic) into a dedicated knowledge base. :contentReference[oaicite:15]{index=15}

## 5. Taking Notes
While reading, reviewing or chatting with a file, you can click **Add Note** to record key insights or inspirations. :contentReference[oaicite:16]{index=16}

---

## 6. Best Practice Tips
- Ask specific, structured questions to get better answers (e.g., “Summarize section 3.2 in 150 words with two direct quotes”). :contentReference[oaicite:17]{index=17}
- Request that answers include page numbers or quote excerpts when working with academic or technical material. :contentReference[oaicite:18]{index=18}
- For scanned or image-based PDFs, OCR quality may impact results — double-check extracted text or numbers. :contentReference[oaicite:19]{index=19}

---

### Want to get started?
1. Upload your first file (PDF, PPT, Word etc).
2. Open the chat interface and ask your first question (e.g., “What are 3 key takeaways from this document?”).
3. Use translation or summarisation features when needed.
4. Organise your files, filter or search them, and use notes to capture insights.
5. Explore follow-up questions and interactions to dig deeper into your document content.
