# Chat with PDF (ChatPDF)

ChatPDF allows you to interact with PDFs and other files as if you were chatting with them.
You can ask questions, summarize sections, extract data, translate content, and perform deep research directly on your documents.

This guide explains the **full workflow**, exactly as it works in the interfaceâ€”from uploading a PDF to chatting, model selection, and error handling.

---

## What You Can Do with ChatPDF

With ChatPDF, you can:

- Upload PDFs and create PDF chat sessions
- View documents alongside a chat panel
- Ask document-aware questions
- Get answers grounded in document content
- Summarize pages, sections, or entire PDFs
- Extract tables, lists, and key data
- Translate documents into other languages
- Chat with multiple PDFs at once
- Manage and delete PDF chat sessions

---

## Interface Overview

The ChatPDF interface is divided into four main areas:

1. **Sidebar Navigation**
   - Access ChatPDF
   - View PDF chats
   - Upload new files
   - Start a new chat

2. **PDF Chat List**
   - Shows uploaded PDFs
   - Displays upload date
   - Allows deleting a PDF chat

3. **PDF Viewer**
   - Displays the selected PDF
   - Page navigation controls
   - Scroll and zoom support

4. **Chat Panel**
   - AI model selector
   - Chat input box
   - Document-aware responses
   - Error and status messages

## 1. Uploading PDFs and Creating PDF Chats

### 1.1 Uploading a PDF

To upload a PDF:

1. Open **ChatPDF** from the sidebar
2. Click **Upload File**
3. Select a PDF from your computer
4. Wait for processing to complete

Once uploaded:
- A new **PDF chat** appears in the left panel
- The PDF opens automatically in the viewer
- A dedicated chat session is created for that file

Each uploaded PDF has its own chat context.

---

### 1.2 What Happens After Upload

After upload, the system performs the following steps:

1. Extracts text from the PDF
2. Applies OCR if the PDF is scanned
3. Splits content into pages and chunks
4. Generates embeddings
5. Stores embeddings in the vector database
6. Indexes content for retrieval

Only after indexing completes can document chat begin.

---

## 2. Viewing and Navigating the PDF

In the PDF viewer, you can:

- Scroll through pages
- Jump to a specific page
- View total page count
- Switch between PDFs

The AI references the currently indexed document, not just the visible page.

---

## 3. Selecting an AI Model

At the top of the chat panel, you can choose an AI model.

### Model Selection Steps

1. Click the model dropdown
2. Select a model (for example, GPT-4o-mini)
3. The selected model is used for all responses in this chat

### Model Selection Tips

- Lightweight models are faster for summaries
- Advanced models are better for deep reasoning
- Model choice does not change document indexing
