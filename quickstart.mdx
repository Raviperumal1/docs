## Basic Structure {#basic-structure}
```ts
GET /api/query?filter={...}&sort={...}&page=1&pageSize=20
```

## Filter Operators {#filter-operators}
### Equality
```json
{"field": "value"}
```

### Not equal
```json
{"field": {"$ne": "value"}}
```

### Range
```json
{"age": {"$gt": 18, "$lt": 50}}
```

### In / Not in
```json
{"status": {"$in": ["active", "pending"]}}
```

## Text Operators {#text-operators}
### Contains
```json
{"name": {"$contains": "john"}}
```

### Starts with
```json
{"email": {"$startsWith": "admin"}}
```

### Full-text search
```json
{"$search": "neural networks"}
```

## Logical Operators
### OR
```json
{"$or": [{"status": "active"}, {"status": "pending"}]}
```

### NOT
```json
{"$not": {"archived": true}}
```

## Sorting & Pagination {#sorting}
```json
{"sort": {"createdAt": "desc"}, "page": 1, "pageSize": 20}
```

## AI Queries {#ai-queries}
### Semantic Search
```json
{"$semantic": "find posts about automation"}
```

### AI Filter Suggestion
```json
{"$aiFilter": "high engagement users"}
```

## Examples {#examples}
```json
{
  "$or": [
    {"category": {"$in": ["ai", "automation"]}},
    {"$search": "workflow optimization"}
  ],
  "published": true,
  "createdAt": {"$gt": "2024-01-01"},
  "sort": {"createdAt": "desc"},
  "page": 1,
  "pageSize": 50
}
```

## FAQ
#### Why is my query returning no results?
Check spelling, remove nested groups, or simplify conditions.

#### Are AI queries slower?
Yes, because they require vector or LLM processing.
