The **Sidebar Navigation** provides centralized access to all core features and account controls in Nexva.

It allows users to switch between features, manage conversations, and access account settings from a single interface.

The sidebar supports both **expanded** and **collapsed** modes and dynamically displays chat history when available.


## Sidebar Layout Overview

The sidebar is divided into four main sections:

1. Header (Branding & New Chat)
2. Feature Navigation
3. Chat History (Conditional)
4. Footer (Plan, Credits, User Menu)


## Header Section

### Nexva Logo & Branding

- Displays the Nexva logo at the top
- Acts as the application identifier
- Clicking the logo may redirect to the home view


### New Chat Button

- Creates a fresh conversation session
- Clears the current chat context
- Available in both expanded and collapsed modes
- Displayed as:
  - **Text + Icon** in expanded mode
  - **Icon only** in collapsed mode

## Chat History Section

The **Chat History** section appears **only when previous conversations are available**.

### What This Section Shows

- List of recent chat conversations
- Each chat displays a title or preview
- Chats are ordered by most recent activity


### How Chat History Works in the Sidebar

1. If chat history exists, it appears between **Feature Navigation** and **Footer**
2. Click a chat to open and continue the conversation
3. The selected chat is highlighted as active

### Plan Indicator

## User Profile Menu

### Avatar, Email & Chevron

- Displays the user avatar or profile icon
- Shows user's avatar, name and email in expanded mode
- Includes a **chevron icon** indicating a dropdown


### Opening the User Menu

1. Click the **User Avatar / Email**
2. A dropdown card opens
